# BL Seller Dashboard - Amazon Sales & Returns Analysis

This dashboard was created as part of the DIO Bootcamp using public data.
It contains sales data for **BL**, a small leather goods company selling on Amazon.
In recent months, the business suffered losses due to order returns, so I built a dashboard to visualize sales and return patterns.

**Important:** As I didn't have Excel installed locally, everything was built using Excel Online.

## Key Insights
- **Total Revenue (2021)**: ₹ 89,177
- **Peak Month**: December (₹ 34,150 — 38% of annual revenue)
- **Return Rate**: 6.4% (11 out of 171 orders returned)
- **Top Product Type**: Handbags & Shoulder Bags
